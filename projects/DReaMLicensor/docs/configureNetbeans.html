Configuring netbeans for License Server

  • Create a new netbeans project by clicking on File->New Project
  • In the new project wizard select Web in Categories and then Web Application with Existing Sources in Projects. Click next.
  • In the Location: text box put in the name of the directory of the license server source code.
    • In the text box below that specify a new project name (e.g. LicenseServer). In the text box below that specify a directory (e.g. /home/asingh/tmp/LicenseServer) that will be used by netbeans to keep build files and netbeans project specific information. Click next
  • Make sure that web pages folder is $LICENSESERVER_SRC_HOME/web, libraries folder is $LICENSESERVER_SRC_HOME/web and source package folder is $LICENSESERVER_SRC_HOME/lib. Click finish
Once the project has been created you can use the netbeans to build and run the project.
